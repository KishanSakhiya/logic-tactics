# logic-tactics
**Question:**
Write a code to print below pattern with some conditions.
```
* * * * * 
* * * * 
* * * 
* * 
* 
```

**Conditions:**
- Use only one loop.(do not use nested loop)
- Take the input from user to get the number of rows

**Sample 1:**
- Input: 3
- Output: 
    ```
    * * * 
    * * 
    * 
    ```

**Sample 2:**
- Input: 5
- Output: 
    ```
    * * * * * 
    * * * * 
    * * * 
    * * 
    * 
    ```

**Sample 3:**
- Input: 10
- Output: 
    ```
    * * * * * * * * * * 
    * * * * * * * * * 
    * * * * * * * * 
    * * * * * * * 
    * * * * * * 
    * * * * *
    ```
